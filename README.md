# ZUBIDA 租必達網站

這版以你提供的新版 Tailwind 風格為主，不混入之前另一套設計。

## 主要頁面
- index.html：首頁
- cooler.html：水冷扇
- tent.html：活動帳篷
- generator.html：發電機

## 舊網址導向
- event-tent.html → tent.html
- water-cooler.html → cooler.html

## 圖片資料夾
圖片放在 pics/

範例：
```html
<img src="pics/你的圖片.jpg" alt="圖片說明">
```
